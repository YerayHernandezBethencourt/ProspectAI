# ProspectAI

**ProspectAI** es una solución de inteligencia artificial avanzada diseñada para facilitar la comprensión de prospectos médicos. Este proyecto permite transformar la información compleja de prospectos médicos en resúmenes claros y puntos relevantes, mejorando la accesibilidad y comprensión de la información médica crítica.

## Descripción

**ProspectAI** utiliza modelos generativos de inteligencia artificial para analizar imágenes de prospectos médicos y extraer información clave de manera efectiva. Los usuarios simplemente toman una foto del prospecto médico, y **ProspectAI** se encarga de:

- **Generar Resúmenes**: Proporciona resúmenes concisos de la información contenida en el prospecto.
- **Destacar Puntos Relevantes**: Extrae y presenta los puntos más importantes, como indicaciones, dosis y efectos secundarios.
- **Responder Preguntas Específicas**: Ofrece respuestas a preguntas detalladas sobre el prospecto, facilitando una comprensión más profunda.

## Características

- **Extracción Automática**: Convierte el contenido de prospectos médicos en texto comprensible mediante OCR y modelos de lenguaje.
- **Resúmenes Claros**: Genera resúmenes que destacan la información crítica de forma rápida y precisa.
- **Interacción Dinámica**: Responde a preguntas específicas sobre el prospecto para una mayor claridad.
